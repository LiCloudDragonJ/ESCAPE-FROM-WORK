# ESCAPE FROM WORK

办公题材俯视角 extraction shooter，Unity 6 开发。

## 快速开始

1. Clone 到本地
2. 用 Unity Hub 打开 `ESCAPE FROM WORK/` 目录（Unity 6000.x）
3. 打开 `Assets/_Project/Scenes/` 下的场景即可运行

## 项目结构

```
├── ESCAPE FROM WORK/     # Unity 项目
│   └── Assets/_Project/  # 游戏代码和资源
├── design/               # 游戏设计文档
│   ├── gdd/              # 设计文档
│   └── registry/         # 实体注册表
├── docs/                 # 参考文档
│   ├── engine-reference/ # Unity 引擎参考
│   ├── superpowers/      # 开发计划
│   └── tooling/          # 工具参考
└── production/           # 会话状态
```

## CCGS 集成

本仓库配合 Claude Code Game Studios (CCGS) 使用。CCGS 本身安装在本地，通过 `/start` 流程加载此项目目录即可。
