# Power System

> **Status**: In Design (Post-MVP)
> **Author**: Claude Code (/design-system)
> **Last Updated**: 2026-07-17
> **Implements Pillar**: 活着的建筑 (#4)

## Overview

Power System 管理写字楼中各楼层的电力状态。停电楼层限制视野（手电筒锥形照明），禁用电梯。关键楼层（如 IT部 27F）需要通过互动恢复电力。MVP 阶段为脚本化触发（关卡设计师手动放置），Post-MVP 可扩展为动态系统。

## Player Fantasy

停电楼层的黑暗中，你只能看到手电筒锥形范围内3米远的地方。打印机故障怪的声音从黑暗中传来，你不知道它在哪个方向。找到服务器机柜，拉下电闸——灯亮了，电梯"叮"的一声，然后你听到保安的脚步声从走廊尽头传来。电力恢复是希望，也是威胁。

## Detailed Design

### 停电效果

| 效果 | 数值 |
|------|------|
| 玩家视野 | 手电筒锥形，30°宽，3m有效距离 |
| 敌人视野 | 不受影响（敌人在黑暗中仍能感知玩家——非真实模拟） |
| 电梯 | 停电楼层电梯不可用 |
| UI | 屏幕边缘暗角效果 + 手电筒锥形高亮 |

### 电力恢复

| 方式 | 描述 |
|------|------|
| 发电机 | 每层有一个备用发电机（随机位置），互动后恢复该层电力 |
| 服务器机柜（27F） | 需要找到3个机柜依次拉闸，全部完成后恢复 |
| 剧情触发 | 1F CEO半身击败后 → 电梯系统全局重启 → 50F折返 |

### 楼层电力状态

| 楼层 | 默认状态 | 特殊 |
|------|---------|------|
| 50F | 有电 | 教学关正常 |
| 27F IT部 | 停电 | 需恢复3个服务器机柜 |
| 随机楼层 | 30%概率停电 | 每次进入时随机判定 |
| 安全楼层 | 有电 | 安全楼层必有电 |

### 手电筒

| 属性 | 值 |
|------|-----|
| 锥形角度 | 30° |
| 有效距离 | 3m |
| 衰减 | 线性，3m处亮度为0 |
| 电池 | 无限（MVP） |
| 开关 | 自动（黑暗楼层自动开启） |

## Formulas

```
停电判定（随机楼层）:
isBlackout = Random.value < 0.30  // 30%概率

手电筒亮度:
intensity = 1 - (distance / 3.0)  // clamped to [0, 1]
```

## Edge Cases

1. **停电楼层 + 战斗** → 手电筒锥形外的敌人不可见，但 auto-aim 仍能锁定（锁定距离减半）。
2. **电力恢复后** → 全层亮灯 + 电梯可用 + 引发噪音事件（敌人向发电机位置移动）。
3. **手电筒穿墙** → 不做光线遮挡（MVP），光线穿透墙体但亮度衰减。
4. **多个发电机** → 每层只有一个有效发电机位置，互动后该层永久恢复。

## Dependencies

| 系统 | 方向 | 类型 | 接口 |
|------|------|------|------|
| Floor Generation | 上游 | Hard | 发电机位置，服务器机柜位置 |
| Elevator & Stairs | 下游 | Hard | 电力状态控制电梯可用性 |
| Enemy AI | 下游 | Soft | 电力恢复噪音吸引敌人 |

## Tuning Knobs

| 参数 | 默认值 | 安全范围 | 描述 |
|------|--------|---------|------|
| blackoutChance | 0.30 | 0.10–0.50 | 随机楼层停电概率 |
| flashlightConeAngle | 30° | 15–60° | 手电筒锥形角度 |
| flashlightRange | 3m | 2–5m | 手电筒有效距离 |
| autoAimRangeInBlackout | 0.5x | 0.3–0.7x | 停电时自动瞄准距离倍率 |

## Acceptance Criteria

1. GIVEN 进入停电楼层, WHEN 加载完成, THEN 手电筒锥形可见, 屏幕边缘暗角
2. GIVEN 玩家靠近发电机, WHEN 按E交互, THEN 电力恢复, 全层亮灯
3. GIVEN 27F IT部, WHEN 拉下3个服务器机柜电闸, THEN 电力恢复
4. GIVEN 停电楼层, WHEN 查看电梯面板, THEN 该楼层标记"电力故障"
5. GIVEN 电力恢复, WHEN 噪音触发, THEN 敌人向发电机位置移动
