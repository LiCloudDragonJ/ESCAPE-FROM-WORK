# ESCAPE FROM WORK — 概念设计草案

## 概述

**ESCAPE FROM WORK** 是一款俯视角 2.5D PvE 搜打撤游戏（搜索-战斗-撤离）。
玩家扮演一头牛或一匹马（Q版动物角色），从一栋写字楼的第50层逐层向下突破，
逃出被怪物占领的窒息办公室文化，最终从1楼大堂逃出生天。

## 核心玩法循环

```
茶水间基地 → 选择目标楼层 → 楼层内Raid(搜刮+战斗+撤离) → 带回资源 → 基地建造
     ↑                                                                      ↓
     └────────────────── 主线推进（关键道具解锁） ←──────────────────────────┘
```

## 楼层结构

- **混合式**：自由选层进入 + 主线向下推进
- 50层→1层单向旅程，5-8层为MVP
- 每层/每几层有独特的部门主题（财务部、IT部、HR部、开放办公区等）
- 电梯：安全楼层间快速往返，但会发出响动吸引怪物
- 楼梯间：慢但安全，每次上下楼有过场动画
- 电力系统：部分楼层停电需拉闸恢复，有电才能用电梯
- 越底层的员工职级越低、怨念越高、战斗力越强
- 保安在每层随机刷新

## 美术风格

- **玩家**：Q版动物（牛/马可选），2-3头身，西装/工服
- **敌人**：办公室文化具象化的怪物（KPI丧尸、PPT怨灵、邮件幽灵、会议恶魔等）
- **氛围**：明亮的办公环境→逐渐阴森的底层

## 战斗系统

- **操作**：WASD移动 + 鼠标瞄准射击 + 闪避翻滚 + 交互搜刮
- **双模式瞄准**：
  - 自动瞄准：省心但不够精准，容易因操作粗糙而受反击
  - 手动瞄准：精准但需要技术，上限更高
- **掩体系统**：办公桌/文件柜/饮水机天然掩体，贴墙自动掩体
- **电力/光照**：停电楼层手电筒锥形视野，恢复供电后全层亮灯

## 武器系统

MVP 10-15种武器，分为两类：

### 办公用品改造（A类）
- 订书机手枪、键盘霰弹枪、碎纸机锯、投影仪射线枪、网线鞭、马克杯投掷器

### 脑洞武器（C类）
- PPT发射器、会议邀请函法杖、KPI报表重锤、咖啡因爆发、邮件炸弹

### 改装系统（MVP简化）
每个武器2-3改装槽：瞄准具、弹药转化、特殊配件

## 基地系统

- **据点**：每层茶水间/休息区可设临时基地
- **设施**：饮水机（回血）、休息沙发（回体力）、储物柜（仓库）、改造台、医疗角、情报板、交易终端、咖啡机（增益道具）
- **资源**：回形针(基础货币)、打印纸(弹药/情报)、咖啡豆(增益)、U盘(稀有配方)、工牌(Boss掉落/楼层权限)
- **核心规则**：建设消耗不可退回，可通过消耗资源搬迁基地，NPC帮忙在安全楼层间转运物资
- **安全楼层**：完全清扫后成为安全区，物资容器不刷新，但敌人不定期刷新（供卡关刷怪）

## 敌人系统

MVP 3-5种普通怪物 + Boss：
- KPI丧尸（近战）、PPT怨灵（远程/遮挡视野）、邮件幽灵（群体自爆）、会议恶魔（减速结界）
- 保安（随机刷新/高防致盲）、精英保安队长（低层/冲锋）
- Boss：经理（中层/二阶段变身）、CEO（最终Boss/召唤+控制）

## 死亡惩罚

- **B+C混合**：安全楼层死亡轻罚（NPC收尸），危险楼层死亡重罚（血本无归）
- 死亡后装备可在保险箱"打工赎装备"——付费赎回，没钱去安全楼层刷怪还债

## 通关条件

- **B+C混合**：关键道具解锁推进节点（IT层拿服务器卡、HR层拿通行证），但最终突破时机由玩家自定
- 准备充分后一次性冲向1楼大堂完成终极逃脱

## 叙事

- **环境叙事为主** + **轻度主线** + **部分深度**
- 故事藏在办公便签、电脑邮件、墙上的标语里
- 核心故事：揭露公司黑幕→逃出生天

## 联机

- 先单机MVP，后续考虑合作PvE或PvPvE

## 开发

- **引擎**：团结引擎 1.9.3（Tuanjie Engine，Unity中国版）
- **体量**：MVP 5-8层 / 10-15武器 / 3-5敌人 / 可通关

- 《致命公司》(Lethal Company) — 氛围、怪物设计
- 《哈迪斯》(Hades) — 渐进式成长、叙事节奏
