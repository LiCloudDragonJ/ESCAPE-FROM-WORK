# Base Building System

> **Status**: In Design (Post-MVP)
> **Author**: Claude Code (/design-system)
> **Last Updated**: 2026-07-17
> **Implements Pillar**: 赌徒时刻 (#3), 下坠之旅 (#2)

## Overview

Base Building System 定义 ESCAPE FROM WORK 中茶水间基地的建造、升级和搬迁规则。茶水间是玩家在写字楼中的安全据点——每 5 层一个，可设为临时基地。基地包含 4 个可升级设施（改造台、医疗角、情报板、咖啡机），每层安全后解锁该层独有功能。资源投入不可退回，搬迁有成本。

## Player Fantasy

茶水间是你在这栋疯了的写字楼里唯一的喘息空间。这里没有敌人，没有 KPI，只有你和其他幸存者。你在改造台上敲敲打打把订书机改成手枪，在咖啡机前冲一杯能让你跑得更快的浓缩咖啡，在情报板上研究下一层的敌人分布。墙上挂满了前人的工牌——提醒你每次离开这个房间都是一场赌局。

## Detailed Design

### Core Rules

1. 每 5 层有一个茶水间（50F, 45F, 40F...），共 10 个茶水间。
2. 玩家可选任意已发现的茶水间作为基地。初始基地 = 50F 茶水间。
3. 基地建造消耗回形针（不可退回）。
4. 搬迁基地有成本（旧基地投入 × 40% 损失）。
5. 每层安全后解锁该层独有功能。

### 设施

#### 改造台 (Workbench)

| 等级 | 成本 | 功能 | 解锁条件 |
|------|------|------|---------|
| Lv1 | 50 回形针 | 武器改装（瞄准具槽） | 初始可用 |
| Lv2 | 200 回形针 | 解锁弹药转化槽 | 发现任意 IT 部楼层 |
| Lv3 | 500 回形针 | 解锁特殊配件槽 | 改造 5 把武器后 |
| 重制 | 武器基础成本 × 0.5 | 重制已解锁蓝图的基础武器 | 初始可用 |

#### 医疗角 (Medical Corner)

| 等级 | 成本 | 功能 | 解锁条件 |

| Lv2 | 150 回形针 | 制作医疗包（恢复 50 HP） | 使用 10 个绷带后 |
| Lv3 | 400 回形针 | 制作兴奋剂（临时 HP 上限 +20%） | 击败任意 Boss 后 |

| 等级 | 成本 | 功能 | 解锁条件 |
|------|------|------|---------|

| Lv3 | 350 回形针 | 显示隐藏物资位置 + 推荐路线 | 发现 3 个隐藏房间后 |

#### 咖啡机 (Coffee Machine)

| 等级 | 成本 | 功能 | 解锁条件 |
|------|------|------|---------|
| Lv1 | 20 回形针 | 制作速溶咖啡（移速 +10%，60s） | 初始可用 |
| Lv2 | 120 回形针 | 制作浓缩咖啡（攻速 +15%，60s） | 消耗 10 个咖啡豆 |
| Lv3 | 300 回形针 | 制作特调咖啡（移速+15% + 攻速+15%，90s） | 收集 3 种不同咖啡豆 |

| 楼层 | 解锁 | 描述 |

| 15F 法务部 | **合同审查** | 减少搬迁成本 50% |

### 基地搬迁

```
搬迁消耗 = 旧基地已投入资源 × 0.40（可被法务部解锁降至 0.20）
实际保留 = 旧基地已投入资源 × 0.60（自动转为新基地建造进度）

步骤:
1. 在新茶水间选择"设为主基地"
2. 确认搬迁消耗
3. 旧基地设施拆除（动画: 打包箱子）
4. 新基地设施重建（保留 60% 投入）
5. 搬迁完成 — 新基地可用
```

### 储物箱

| 属性 | 值 |
|------|-----|
| 尺寸 | 8×10 = 80 格（可升级） |
| 跨角色共享 | ✅（所有幸存者共用） |
| 死亡不掉 | ✅ |
| 升级成本 | +1 行（10 格）= 200 回形针，最多 +4 行（共 120 格） |

## Formulas

### 设施升级成本

```
upgradeCost[level] = baseCost × (1 + level × 0.5)
示例: 改造台 Lv1=50, Lv2=200 (=50×4), Lv3=500 (=50×10)
```

### 搬迁成本

```
relocationCost = totalInvested × relocationRate
totalInvested = sum(upgradeCost for all facilities at current base)
retainedValue = totalInvested × (1 - relocationRate)
```

### 医疗品制作

```
bandageCost = 10 paperclips + 2 printerPaper → 恢复 25 HP
medkitCost = 25 paperclips + 5 printerPaper + 1 cloth → 恢复 50 HP
stimulantCost = 50 paperclips + 3 coffeeBean + 1 USB → 临时 HP 上限 +20% (持续 180s)
```

### 咖啡制作

```
instantCoffee = 5 paperclips → 移速 +10%, 60s
espresso = 10 paperclips + 1 coffeeBean(fresh) → 攻速 +15%, 60s
specialBlend = 20 paperclips + 1 coffeeBean(fresh) + 1 luxury → 移速+15% + 攻速+15%, 90s
```

## Edge Cases

1. **搬迁到未完全安全的楼层**: 不允许。基地只能在安全楼层（完全清扫）设置。
2. **搬迁中基地楼层被重新入侵**: 安全楼层 4 小时后刷新敌人。如果基地楼层被敌人再次进入 → 基地设施暂停使用 → 需重新清扫。
3. **设施升级中退出游戏**: 升级立即完成（无建造时间），退出不影响。
4. **多个茶水间可选时**: 自动选择最近解锁的地图最低层的茶水间作为推荐基地。
5. **储物箱满**: 阻止存入，提示"储物箱已满"。
6. **NPC 交易价格**: 买价 = 物品基础价值 × 1.2，卖价 = 物品基础价值 × 0.5。
7. **蓝图升级后武器数据更新**: 已拥有的该型号武器自动升级（重新计算数值）。

## UI Requirements

- 基地主界面：4 个设施图标（改造台/医疗角/情报板/咖啡机）+ 储物箱 + 武器架 + 公告板
- 设施升级面板：显示当前等级、升级效果预览、所需材料
- 储物箱界面：复用 LootContainerUI（三栏面板——装备 | 背包 | 储物箱）
- 搬迁确认弹窗：显示消耗 + 保留资源 + 确认/取消

## Acceptance Criteria

1. GIVEN 玩家在茶水间, WHEN 点击改造台, THEN 显示改装界面和重制选项
2. GIVEN 改造台 Lv1, WHEN 花费 200 回形针, THEN 升级到 Lv2（解锁弹药转化槽）
3. GIVEN 基地在 50F, WHEN 搬迁到 45F, THEN 消耗 40% 投入资源, 保留 60% 设施
4. GIVEN 储物箱有空间, WHEN 玩家存入物品, THEN 物品从背包转移到储物箱
5. GIVEN 医疗角 Lv1, WHEN 消耗材料制作绷带, THEN 绷带出现在背包中
6. GIVEN 情报板 Lv1, WHEN 查看下一层信息, THEN 显示敌人类型（不显示位置）
7. GIVEN 玩家死亡后选新角色, WHEN 进入基地, THEN 所有设施和储物箱完好
8. GIVEN 27F IT 部安全, WHEN 使用服务器消耗 U 盘, THEN 指定蓝图武器数值提升
