## 1. 概要

**ESCAPE FROM WORK** 是一款第三人称越肩视角搜打撤游戏。核心玩法是正经的搜刮-战斗-撤离循环，死亡有代价，撤离有心跳。游戏以写字楼办公室为背景，用"打工人牛马"的设定提供共鸣和梗——但射击手感、战术决策、装备风险都是硬核搜打撤的标准。架构预留第一人称切换可能。

**核心差异化**：搜打撤品类中首个带有明确叙事结局的战役结构——你的每一次Raid都累积向最终逃亡推进。

---

## 2. 设计支柱

1. **办公室即武器库** — 一切压迫你的东西都变成武器。即兴改造、废物利用
2. **下坠之旅** — 50→1的垂直旅程是体验的脊梁，每个楼层向下是永久进度
3. **赌徒时刻** — 搜打撤的紧张感真实存在。死亡有牙齿。"再下一层？"永远是个问题
4. **活着的建筑** — 办公楼不是背景。电力、安保、部门差异创造动态局面

### 反支柱（这游戏不是）

- 不是换皮鸭科夫
- 不是Roguelike
- 不是纯搞笑游戏
- 不是无限服务型游戏
- 不依赖多人联机

---

## 3. 玩家体验

**基调**：正经搜打撤 + 打工人背景的共鸣梗。游戏的紧绷感、死亡惩罚、枪械手感都按正经搜打撤来做——玩家真的会紧张、真的会心疼装备。幽默不来自玩法本身，而是来自"订书机当枪使""KPI报表当锤子抡""被会议恶魔追杀"这些打工人看了会心一笑的设定。玩法严肃，皮肤荒诞。

**核心幻想**：你被压榨了十年。今天你拿起订书机，杀出一条血路。CEO在1楼——让他亲自签你的离职单。

---

## 4. 核心玩法循环

```
茶水间基地 → 选择目标楼层 → 楼层内Raid(搜刮+战斗+撤离) → 带回资源 → 基地建造
     ↑                                                                      ↓
     └────────────── 主线推进（关键道具解锁特殊楼层） ←───────────────────────┘
```

一次Raid预期10-20分钟，基地建设3-5分钟，形成"出击→建造→再出发"的循环。

## 5. 楼层结构

### 概览

- 50层写字楼，全部在地上（无地下室）
- **10个手设计特殊楼层**（叙事节点、Boss战、关键道具）
- **40个随机生成普通楼层**（3 种布局原型：RingStandard 45% / OpenPlan 30% / Cellular 25%）
- 每层 100m × 80m，核心筒（16m×12m）+ 环形走廊结构
- 8 个消防楼梯间锚点（四边各 2），入口随机，出口 BFS 最远
- 茶水间每 5 层一个（50F, 45F, 40F...），靠近入口楼梯

### 封闭楼层叙事

电梯和楼梯跳过封锁的中间楼层。电梯屏幕显示"39层-28层：封锁中 / 27层-6层：结构损坏"——暗示CEO用物理手段锁住了中间层。既是叙事钩子，也为未来扩展预留空间。

| 元素 | 随机方式 | 频率 |

| 楼层地图布局 | 模块随机拼接 | 每次新游戏，该存档内固定 |

| 敌人刷新点 | 固定区域内随机位置 | 每次进入楼层刷新（含安全楼层） |
| 茶水间 | 固定位置 | 不变 |
| 电梯/楼梯 | 固定位置 | 不变 |
| 关键任务道具 | 固定位置 | 不变 |

## 7. 楼层状态

### 安全楼层

- 条件：完全清扫（消灭该层所有敌人）
- 容器物资刷新：每4小时（现实时间）刷新
- 同一安全楼层24小时内连续探索 → 物资数量和品质逐次递减
- 24小时后（从第一次进入起算）递减重置
- 敌人持续刷新（供卡关刷怪）
- NPC幸存者逐渐出现，提供支线任务
- 每层安全后解锁该层的独有功能

### 危险楼层

- 未清扫或未完全清扫的楼层
- 死亡惩罚重
- 物资完整

### 茶水间据点

每层茶水间可设为临时基地。建设消耗不可退回。可搬迁。

| 设施 | 功能 | 升级方向 |
|------|------|----------|
| 改造台 | 武器改装/重制 | 解锁更多槽位 |
| 医疗角 | 制作回血道具 | 更强医疗品 |
| 情报板 | 显示下一层敌人分布 | 隐藏物资/路线 |
| 咖啡机 | 制作增益道具 | 效果强化 |

### 每层安全后独有功能

| 楼层 | 解锁 |
|------|------|
| 50F CEO层 | 情报终端：查看全楼电梯/电力状态 |
| 41F 市场部 | 交易NPC：买卖物资 |
| 27F IT部 | 服务器：消耗U盘升级蓝图 |
| 3F 保安部 | 监控室：查看任意一层敌人分布热力图 |

### 基地搬迁

```
搬迁消耗 = 旧基地已投入资源 × 40%

```

## 13. 死亡系统

### 叙事设定

每次死亡意味着这个角色永远留在了公司里——又一个过劳死的牛马。幸存者团体在茶水间召开紧急会议，选出一位新代表，继承前辈的遗志和基地资源，继续向下推进。

"上一个牛马倒在了27楼。他是带着3个U盘回来的——服务器房里的秘密被他带回来了。我们得派下一个下去。"

### 角色

- 每次死亡后玩家从幸存者中选一个新角色（牛/马外观可选，属性相同）
- 前一个角色的尸体留在死亡楼层，装备在尸体上
- 茶水间墙上多一个**纪念牌**——记录每个死去的角色：名字、死亡楼层、带回的物资

### 死亡掉落

| 死亡地点 | 丢失 | 保留 |
|----------|------|------|
| 安全楼层 | 本次搜刮的全部物资 + NPC收尸小费 | 装备武器、基地进度 |
| 危险楼层 | 本次搜刮物资 + 身上装备的武器和消耗品 | 基地进度、保险柜内物品 |

### 尸体回收（新角色操作）

- 新角色可返回死亡地点捡回前辈的尸体装备
- 保安可能已巡逻经过——尸体装备可能被搜走一部分
- 安全楼层死亡：NPC同事帮忙收尸取回装备，付固定小费
- 捡回的装备完好取回

### 装备恢复

- 基地改造台可用回形针重新制作已解锁蓝图的基础武器
- 部分稀有武器蓝图需在楼层中找到，解锁后永久可用
- 幸存者团体日常储备2-3套备用装备

### 基地继承

- 基地所有设施、解锁蓝图、安全楼层进度 → 全部保留
- 新角色继承前辈的一切，除了身上的装备
- 纪念牌是唯一的损失——但也是荣耀

### 保险柜

每层茶水间有一个保险柜。存放的物品死亡不掉。存取消耗回形针。容量有限。

## 14. 电梯与楼梯

### 电梯

| 规则 | 设定 |
|------|------|
| 条件 | 仅安全楼层间 / 目标楼层电力已恢复 |
| 速度 | 即时到达 |
| 风险 | 到达时"叮"声 → 当前楼层所有敌人向电梯移动15秒 |
| 战术 | 可设伏——让电梯先到，从楼梯绕后 |

### 楼梯

| 规则 | 设定 |
|------|------|
| 条件 | 所有楼层始终可用 |
| 速度 | 每层3秒动画（可跳过） |
| 风险 | 每下一层15%概率遇敌，跨度越大概率越高 |

### 消防通道

- 每层固定位置（与入口普通楼梯呈对角线）
- 只能从内部推开（只能撤离，不能进入）
- 快速撤离路线，但需穿越未探索区域到达

### 决策三角

```
进入：    普通楼梯（安全可控）
撤离A：  原路返回（已清区域，但可能有增援）
撤离B：  消防通道（快，但需要冲到对角线）
撤离C：  电梯（仅安全楼层，确定风险可设伏）
```

## 15. 电力系统

- MVP：脚本化触发（关卡设计师手动放置），非全局系统
- 停电楼层：手电筒锥形视野，黑暗中可能被伏击
- IT部(27F)：需找到3个服务器机柜拉闸恢复供电
- 电力恢复后：全层亮灯 + 电梯可用 + 电梯噪音引来敌人

## 17. 关键道具

### 工牌（狗牌系统）

工牌是每个角色的身份标识——类似塔科夫的狗牌。所有敌人和角色都有一张工牌。

**工牌信息**：
- 姓名
- 部门/职位
- 员工编号
- 在职年限

**工牌来源**：

| 来源 | 工牌类型 | 用途 |
|------|----------|------|
| 普通敌人掉落 | 普通工牌 | 卖回形针，品级越高越值钱 |
| 精英/Boss掉落 | 特殊工牌 | 解锁楼层权限（永久flag）+ 高价值收藏品 |
| 前人角色尸体 | 前人遗物 | 回收后挂在茶水间纪念墙上 |

**前人遗物——纪念墙**：
- 每个死去的前人角色，尸体上有一张工牌
- 新角色回收后，工牌挂在茶水间纪念墙上
- 纪念墙显示：姓名、死亡楼层、死因、带回的物资价值
- 工牌本身也可以选择卖掉（纯属个人选择——留下纪念还是换资源）

**工牌交易**：
- 普通工牌按品级卖给NPC换取回形针
- 品级 = 敌人的等级/部门，越底层的敌人职级越低但工牌越"值钱"（怨念越深，幸存者越愿意高价收购作为纪念）
- Boss特殊工牌不可出售——永久进度物品

## 18. 叙事

### 故事前提

你是黑心集团的老牛员工，干了十年。今天CEO发了全员邮件："公司即日起实施永久闭环办公制度。所有人不得离开大楼。祝工作愉快。"当你抬头，同事们已经开始变成怪物。你从50楼杀到1楼，击败了CEO半身——却发现那也不是本体。真正的CEO还在50楼等着你。你折返回去，这次，让他亲自签你的离职单。

### 敌人起源

CEO启动"高效闭环管理系统"——本质是诅咒契约。员工的工作执念具象化成怪物。CEO和某个古老存在做了交易：用整栋楼的员工作为代价，换取公司永远运转。

### 信息发现弧线

| 楼层 | 阶段 | 揭示 |
|------|------|------|
| 50F CEO办公室 | 起点 | 新手教学：收到全员邮件，遭遇CEO假身（分身），开始逃亡 |
| 41F 市场部 | 裂缝 | 备忘录：CEO雇了"高效闭环"咨询公司 |
| 35F 财务部 | 裂缝 | 巨额咨询费，项目代号"永动机"——不用下班、不要工资、不会辞职的劳动力 |
| 27F IT部 | 深入 | 服务器日志："永动机"是员工意识上传计划 |
| 21F HR部 | 深入 | 人事档案：所有人"自愿"签了隐藏条款 |
| 15F 法务部 | 关键 | 合同原件："永动机"是诅咒契约，CEO和某个存在做了交易 |
| 10F 档案室 | 关键 | 这栋楼建在古老土地上，每几十年收割一批人 |
| 4F 培训部 | 高潮前 | 你的入职档案——你也签了 |
| 3F 保安部 | 终局前 | 保安是知情的看守 |
| 1F 大堂 | 假终点 | CEO半身战：击败半身后发现——这也不是本体 |
| → 电梯重新激活 → | 折返 | 50F的本体感应到半身被毁，电梯系统重启 |
| 50F CEO办公室 | 终极战 | CEO本体：三阶段终极Boss战，三个结局 |

### 结局

1F击败CEO半身后，真相揭露：CEO本体在50楼。电梯重新激活，你必须折返回顶层。

在50F击败CEO本体后，三个结局：

1. **撕碎契约（真结局）**：拒绝CEO的交易提议，彻底摧毁诅咒契约。大楼开始崩毁，你在倒计时中冲向旋转门。阳光洒下来——你是第一个走出这栋楼的人。幸存者们在外面等你。

2. **代替CEO签字（契约结局）**：接受CEO的条件——你成为新的"永动机"核心。画面定格在你坐在CEO椅子上，下一个"你"从50楼醒来。循环继续。

3. **同归于尽（牺牲结局）**：在CEO濒死时拉他一起跳楼。大楼诅咒随着CEO的死亡解除，所有幸存的同事走出大楼。你的纪念牌挂在茶水间最显眼的位置。
