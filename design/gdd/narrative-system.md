# Narrative System

> **Status**: In Design (Post-MVP)
> **Author**: Claude Code (/design-system)
> **Last Updated**: 2026-07-17
> **Implements Pillar**: 下坠之旅 (#2)

## Overview

Narrative System 通过 10 个特殊楼层的文本/音频/环境叙事传递 ESCAPE FROM WORK 的故事——从 50F 收到全员邮件开始，到 1F 击败 CEO 半身后折返 50F 决战 CEO 本体。叙事是线性的（信息发现弧线按楼层推进），但交付方式多样化（备忘录、终端、物品描述、NPC对话、环境场景）。

## Player Fantasy

你不是在"读故事"——你是在"拼故事"。在41F市场部发现CEO雇了"高效闭环"咨询公司的备忘录；在35F财务部看到巨额咨询费转账记录；在27F IT部服务器日志里发现"永动机"的真正含义。每一层都是拼图的一块。到10F档案室时，你终于明白这栋楼的真相。

## Detailed Design

### 信息发现弧线（更新版 — CEO三层结构）

| 楼层 | 阶段 | 揭示 | 交付方式 |
|------|------|------|---------|
| 50F | 起点 | 收到全员邮件；遭遇CEO假身；逃脱后CEO嘲讽 | 终端邮件 + CEO假身Boss战对话 |
| 41F | 裂缝 | 备忘录：CEO雇了"高效闭环"咨询公司 | 办公桌搜刮品 |
| 35F | 裂缝 | 巨额咨询费，代号"永动机" | 财务报告物品 |
| 27F | 深入 | 服务器日志："永动机"是员工意识上传 | 服务器终端交互 |
| 21F | 深入 | 人事档案：所有人"自愿"签了隐藏条款 | 文件柜搜刮品 |
| 15F | 关键 | 合同原件：诅咒契约 | 保险柜物品 |
| 10F | 关键 | 这栋楼的历史——每几十年收割一批人 | 档案室终端 |
| 4F | 高潮前 | 你的入职档案——你也签了 | 培训部终端 |
| 3F | 终局前 | 保安队长对话：CEO"不止一个"（伏笔） | Boss战对话 |
| 1F | 假终点 | CEO半身战；击败后发现不是本体 | Boss战对话 + 电梯重启动画 |
| 50F | 终极 | CEO本体战；三个结局选择 | Boss战对话 + 结局选择 |

### 叙事交付方式

| 方式 | 描述 |
|------|------|
| 终端邮件 | 可交互的电脑屏幕，显示文本/UI |
| 搜刮品叙事 | 特定物品带有 lore 文本（如"财务报告"物品描述） |
| Boss对话 | Boss战前后播放简短对话（文字气泡 + 配音预留） |
| 环境场景 | 特殊楼层的视觉布置暗示故事（如15F合同柜、10F档案架） |
| NPC对话 | 茶水间幸存者NPC谈论世界观（老王、小美、大刘、老周） |

### 结局系统

| 结局 | 条件 | 50F CEO本体战后 |
|------|------|----------------|
| 撕碎契约（真结局） | 收集全部10个特殊楼层的叙事物品 | 拒绝CEO交易 → 大楼崩毁 → 逃出旋转门 |
| 代替签字（契约结局） | 未收集全叙事物品 | 接受CEO条件 → 成为新的"永动机"核心 |
| 同归于尽（牺牲结局） | 任意条件，在CEO本体HP<5%时选择跳楼 | CEO死亡 → 诅咒解除 → 纪念牌 |

## Formulas

```
叙事完成度 = 已收集叙事物品数 / 10
叙事完成度 = 1.0 → 解锁真结局条件
叙事完成度 < 1.0 → 仅契约结局和牺牲结局可选
```

## Edge Cases

1. **跳过特殊楼层** → 可通过楼梯跳过。该楼层的叙事物品永远丢失（该 run 内）。
2. **叙事物品被丢弃/卖出** → 一旦拾取过即计入完成度（不需要一直携带）。
3. **新角色继承叙事进度** → 叙事完成度是元进度（跨角色持久化）。

| Floor Generation | 上游 | Hard |

## Acceptance Criteria

1. GIVEN 玩家首次到达特殊楼层, WHEN 交互叙事物件, THEN 显示 lore 文本
2. GIVEN 收集全部10件叙事物品, WHEN 击败CEO本体, THEN 真结局可选
3. GIVEN 未收集全叙事物品, WHEN 击败CEO本体, THEN 仅契约/牺牲结局可选
