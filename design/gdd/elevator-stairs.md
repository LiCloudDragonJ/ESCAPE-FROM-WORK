# Elevator & Stairs

> **Status**: In Design (Post-MVP)
> **Author**: Claude Code (/design-system)
> **Last Updated**: 2026-07-17
> **Implements Pillar**: 下坠之旅 (#2), 赌徒时刻 (#3)

## Overview

Elevator & Stairs System 定义楼层间的三种移动方式：电梯（安全楼层间即时到达，但有噪音代价）、楼梯（始终可用，每层3秒，有遇敌概率）、消防通道（只能撤离不能进入，位于对角线）。三种方式构成"决策三角"——玩家每次移动都需要在速度、安全、风险之间权衡。

## Player Fantasy

电梯"叮"的一声是这栋楼里最恐怖的声音——你知道接下来15秒，这层楼的所有敌人都会向电梯移动。楼梯是可靠的，但每下一层都可能撞上巡逻的保安。消防通道是救命稻草——冲过未探索区域，撞开门，逃出生天。

## Detailed Design

### 电梯

| 属性 | 值 |
|------|-----|
| 条件 | 仅安全楼层间 / 目标楼层电力已恢复 |
| 到达速度 | 即时（0秒加载） |
| 风险 | 到达时"叮"声 → 当前楼层所有敌人向电梯移动，15秒后到达 |
| 噪音半径 | 全楼层（到达通知所有敌人） |
| 噪音持续时间 | 15秒 |
| 战术 | 可设伏——让电梯先到，从楼梯绕后 |
| UI | 楼层选择面板（仅显示可用楼层） |

### 楼梯

| 属性 | 值 |
|------|-----|
| 条件 | 所有楼层始终可用 |
| 速度 | 每层3秒过场动画（可跳过） |
| 遇敌概率 | 基础15%/层，跨度越大越高 |
| 风险 | 楼梯间可能刷新巡逻保安 |
| UI | 楼层指示牌 + "下一层"按钮 |

```
楼梯遇敌公式:
encounterChance = 0.15 × floorsSkipped × (1 + 0.05 × floorsSkipped)

1层: 15.75%
3层: 51.75%
5层: 94.5%  (几乎必然遇敌)
```

### 消防通道

| 属性 | 值 |
|------|-----|
| 位置 | 与入口普通楼梯呈对角线 |
| 方向 | 只能从内部推开（只能撤离，不能进入） |
| 速度 | 即时到达基地（或最近的茶水间） |
| 代价 | 需要穿越未探索区域到达 |
| 风险 | 路上可能遭遇敌人 |

### 决策三角

```
进入楼层:    普通楼梯（安全可控，可回退）
撤离A:      原路返回（已清区域，但可能有增援）
撤离B:      消防通道（快，但需穿越未探索区域）
撤离C:      电梯（仅安全楼层，确定风险但可设伏）
```

### 50F折返机制（CEO三层结构）

击败1F CEO半身后：
1. 电梯系统重新激活
2. 玩家通过电梯返回50F
3. 50F不再是安全楼层——CEO本体等待中
4. 电梯直达（跳过中间楼层生成）

## Formulas

```
电梯噪音吸引力 = 全楼层 × 15秒
楼梯遇敌概率 = 0.15 × floorsSkipped × (1 + 0.05 × floorsSkipped)
消防通道撤离时间 = 即时（0秒加载）+ 到达消防通道的时间

楼层选择 UI:
  安全楼层: 绿色高亮 + "电梯可用"标记
  危险楼层: 灰色 + "需先清扫"
  封锁楼层: 红色 + 封锁原因文字
```

## Edge Cases

1. **电梯目标楼层电力中断** → 电梯不可用，提示"目标楼层电力故障"。
2. **楼梯间被敌人占据** → 进入楼梯时触发战斗，不能跳过。
3. **消防通道在未探索区域** → 需要先到达对角线位置才能使用。
4. **所有安全楼层都被封锁** → 只能使用楼梯（电梯条件不满足）。
5. **50F 折返时电梯** → 不需要电力检查，剧情强制激活。

## Dependencies

| 系统 | 方向 | 类型 | 接口 |
|------|------|------|------|
| Floor Generation | 上游 | Hard | 楼梯间/电梯位置，安全楼层状态 |
| Power System | 上游 | Hard | 电梯需要电力 |
| Enemy AI | 下游 | Soft | 电梯噪音吸引敌人 |

## Tuning Knobs

| 参数 | 默认值 | 安全范围 | 描述 |
|------|--------|---------|------|
| stairBaseEncounterChance | 0.15 | 0.05–0.30 | 楼梯基础遇敌率 |
| stairEncounterScaling | 0.05 | 0.01–0.10 | 跨度加成系数 |
| elevatorNoiseDuration | 15s | 10–30s | 电梯噪音持续时间 |
| stairAnimationDuration | 3s | 1–5s | 每层楼梯过场时长 |

## Acceptance Criteria

1. GIVEN 目标楼层是安全楼层, WHEN 选择电梯, THEN 即时到达, 触发噪音事件
2. GIVEN 目标楼层电力中断, WHEN 选择电梯, THEN 提示不可用
3. GIVEN 玩家通过楼梯下3层, WHEN 到达, THEN 遇敌概率约52%
4. GIVEN 玩家在消防通道门口, WHEN 交互, THEN 即时撤离到基地
5. GIVEN 1F CEO半身被击败, WHEN 使用电梯, THEN 50F可用, 直达折返
